# Description
**The purpose of this repo is to learn Typescript for beginners.**
## Setup the environment
1. First of all we need nodeJs to be installed 
	(link: [nodeJs](https://nodejs.org/en "nodeJs")).
2. Now we need to install 2 packages globally so open your CMD or BASH and run thoses 2 commands: 
	- `npm install -g typescript`
	- `npm install -g ts-node`

## Run this project
1. `git clone https://github.com/mohamedbirali/typescript.git`
2. `cd typescript`
3. `npm i`
4. `npm run start:dev`
